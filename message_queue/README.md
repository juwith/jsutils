How to use  

prepare .
```
$ cd ../queue/
$ ./autogen.sh
$ ./configure 
# or $ ./configure --prefix=${PWD}/install

$ make
$ make install

# if you use prefix when configure
# export PKG_CONFIG_PATH=${PREFIX}/queue/install/lib/pkgconfig
```
